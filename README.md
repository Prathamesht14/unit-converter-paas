# Unit Converter (PaaS Project)

## 📌 Description

This project is a menu-driven unit converter developed in C language.
It performs multiple conversions such as:

* Celsius to Fahrenheit
* Kilometer to Miles
* INR to USD
* SGPA to Percentage
* CGPA to Percentage
* Percentage to Grade
* Grade to Percentage

## ☁️ Cloud Concept

This project is deployed using **Google App Engine (PaaS)** where:

* Code is stored in GitHub
* Executed on cloud platform
* No infrastructure management required

## ⚙️ Technologies Used

* C Programming
* Google App Engine
* GitHub

## ▶️ How to Run Locally

```bash
gcc main.c -o app
./app
```

## 🚀 Deployment

The application is deployed using Google App Engine with a custom runtime (Docker).

## 🎯 Objective

To understand Platform as a Service (PaaS) by deploying and executing applications on the cloud.
